# **App Name**: EarBeatz

## Core Features:

- User Authentication: Secure signup/login using Firebase Authentication.
- Music Library: Browse songs, albums, and artists with categorized metadata and album art.
- Personalized Playlists: Generate mood-based playlists (e.g., Happy, Chill) based on detected mood using tool to guide playlist generation.
- Search and Discovery: Find songs, artists, and playlists with trending and personalized recommendations.
- Social Music Feed: Share favorite tracks and playlists with likes and comments.
- Smart Notifications: Alert users about new releases from their favorite artists.

## Style Guidelines:

- Primary color: Vibrant purple (#000000) to evoke creativity and musicality.
- Background color: Light purple (#822BFE) for a calming and modern feel.
- Accent color: Electric blue (#C85BFF) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, machined feel that works well in both roles.
- Use clean, minimalist icons with a focus on intuitive and recognizable symbols.
- Incorporate smooth transitions and subtle animations to enhance user interaction.
